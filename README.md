\## 📦 Installation

Using npm:

````bash

npm install -g watchman-cli



Or run directly:



npx watchman-cli server.js

🚀 Usage

watchman server.js

📁 Example

watchman app.js



Now, whenever app.js changes → it will automatically restart.
````
